## Getting Started

First, run the development server:

```
Installation and launch

Clone the repository:

git clone https://github.com/myrgorod/Test-Car-Dealer-App.git

Go to the project folder:

cd Test-Car-Dealer-App

Install dependencies:

pnpm install

Start the local server:
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.

## Functionality

The app a car dealer application created using Next.js. The application allow users to filter vehicles by type and model year, and view the results on a separate page.

The app uses as a style dark-mode.
